# 🌦️ City Weather Dashboard

A sleek, modern, and intuitive weather web application that goes beyond simple temperature tracking. Built with a focus on clean design and practical utility, this dashboard provides real-time updates and helpful "What to Wear" advice for cities worldwide.

🔗 **Live Demo:** [https://city-weather-dashboard-nu.vercel.app/](https://city-weather-dashboard-nu.vercel.app/)

## ✨ Features

- **🔍 Smart Search:** Instantly lookup the current weather for any city across the globe.
- **🧣 "What to Wear" Advice:** Dynamic, logic-based recommendations on clothing and gear (e.g., "Bring an umbrella!" or "Wear a heavy jacket") based on current temperature and precipitation.
- **📅 5-Day Forecast:** Planning ahead? View a clean outlook for the next 5 days, filtered for noon-time conditions.
- **🏙️ Quick Suggestions:** A convenient dropdown for major global cities like New York, London, Tokyo, and more.
- **⚡ Real-time Data:** Powered by the OpenWeatherMap API for accurate, up-to-the-minute weather stats.
- **📱 Fully Responsive:** Optimized for desktop, tablet, and mobile viewing.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Modern layout with a professional, clean aesthetic.
- **JavaScript (ES6+):** Asynchronous API fetching and dynamic DOM manipulation.
- **OpenWeatherMap API:** The core engine for weather and forecast data.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/varadbhardwaj2003-commits/City-Weather-Dashboard.git
   ```
2. **Open the project:**
   Simply open `index.html` in your favorite browser, or serve it locally using a tool like Live Server or `python3 -m http.server`.

3. **API Key:**
   The project is pre-configured with an API key for demonstration purposes. For production use, please obtain your own key from [OpenWeatherMap](https://openweathermap.org/api).
